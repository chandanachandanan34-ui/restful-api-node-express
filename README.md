# restful-api-node-express
🚀 Express CRUD API

A simple and clean RESTful API built using Node.js and Express.js that performs full CRUD operations — GET, POST, PUT, PATCH, and DELETE.
This project helped me understand how backend systems handle requests and manage data efficiently.

📌 Features
✅ Create data using POST
📥 Fetch data using GET
🔄 Update full data using PUT
✏️ Update partial data using PATCH
❌ Delete data using DELETE
⚡ RESTful API structure
🌐 Client-server communication using Axios
🛠️ Tech Stack
Node.js
Express.js
Axios
Body-parser

⚙️ Installation & Setup
Clone the repository

Navigate to project folder

Install dependencies
npm install
Run the server
nodemon index.js

or

node index.js
Open in browser
http://localhost:3000
